## Installation

To Install necessary dependencies, run the following commands:  

```
git clone git@github.com:npokamestov/note-taker.git
```
```
npm init
```
```
npm i
```
```
npm install express
```
```
npm install shortid
```
```
npm install -g heroku
```
```
heroku create
```


## Usage
```
npm start	//starts running the server
```
